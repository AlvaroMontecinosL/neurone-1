Below there is an example of a HTML Template for NEURONE. Modals are customizable pop-up content screens (inside the web browser) that can be used in some stages of the NEURONE study. These templates must be written in plain HTML and they can contain Twitter Bootstrap styling, NEURONE translation strings and/or AngularJS controller logic.

NEURONE translation strings must be called inside the template as {{ "mystring" | translate }}, where mystring is a key inside the NEURONE Translation JSON file.

AngularJS controller logic from NEURONE can be used referencing the modal scope object. As a convention, all data passed to the modal must be loaded through the modal.fields object.
